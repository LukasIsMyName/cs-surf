Combat-Surf

Combat surf mode is a mix of basic shooting and surfing. Surfing is a gamemode where a player is "surfing" on a an angular ramp using WASD keys and changing trajectory using a mouse using a smooth movement to keep the velocity.
M3 (Shotgun) is a main weapon in this game mode as it deals the most damage and is easy to use while surfing.
